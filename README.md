# WEATHER DASHBOARD

App made by Ignacio Diaz. Here user can find any weather and 5-day forecast for most cities around the world.

##To create this app Open Weather API was used https://openweathermap.org/

Javascript uses fetch to get information from the API, then it loads all info needed and pushes elements and their content to HTML
